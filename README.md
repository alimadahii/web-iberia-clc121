# web-iberia-clc121
This Web GIS application provides an interactive visualization environment for analyzing:

Baseline industrial structure (CLC121 – 2006)

Industrial/commercial land change (2006–2018)

Machine learning predictions (DV121 probability)

Development Readiness Typology

Spatial covariates (transport, geography, climate, accessibility)

The application is built using:

ArcGIS Online (WebMap)

ArcGIS JavaScript API 4.x

GitHub Pages (hosting)

Machine Learning outputs generated in ArcGIS Pro / Python
